General

Bingocpp is part of the open source package Bingo for performing symbolic regression. Bingocpp contains the c++ implementation of a portion of the code within bingo.

Dependencies

The build process requires:

  • A recent version of cmake
  • git
  • mercurial (hg)

The other follwoing dependencies will automatically be downloaded and installed as part of the cmake build:

  • google test (gtest), as part of the testing suite
  • pybind, for the python bindings
  • eigen, for much of the linear algebra and vectorized math

Tests

Several unit and integration tests will be made upon building. You can run the testing suite by running the "run_tests" executable in the build folder.
